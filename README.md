
# NEXUS VOID 2089

> A Cyberpunk Digital Civilization built with Flutter.
>
> Not a portfolio. Not a landing page.
> A living system.

---

## Overview

**NEXUS VOID 2089** is an experimental cyberpunk portfolio experience that transforms personal identity, projects, skills, and digital memories into an explorable futuristic civilization.

Instead of presenting information through traditional cards and sections, the application creates an immersive environment where visitors navigate dimensions, unlock discoveries, interact with an artificial consciousness, and explore a fictionalized digital world.

The project is built entirely in Flutter and demonstrates advanced UI composition, animation systems, procedural interactions, and narrative-driven user experience design.

---

## Features

### Boot Sequence

A classified system initialization process that gradually reveals access to the world.

* Animated startup protocol
* Sequential activation stages
* Dynamic system messages
* Immersive entry experience

### Dimensional Navigation

The application is organized into multiple interconnected dimensions.

* HOME
* MEMORY
* PROJECTS
* ARCHIVE
* MATRIX
* SIGNALS

Each dimension represents a different aspect of the digital civilization.

### Interactive World Engine

User actions directly influence the environment.

* Pointer tracking
* Scroll-reactive transitions
* Dynamic discovery system
* Achievement progression
* Hidden interactions

### Memory Fragment System

Traditional biography sections are replaced with narrative memory fragments.

* Story-driven identity presentation
* Character-based information delivery
* Cyberpunk world-building

### Project Worlds

Projects are represented as explorable worlds rather than static cards.

* World-based visualization
* Custom atmospheric themes
* Interactive transitions

### Skill Matrix

A visual technology network displaying technical capabilities.

Current matrix includes:

* Python
* C
* JavaScript
* Dart
* Linux
* Go
* Ruby
* Pentesting
* Web Security
* Kotlin
* PHP
* HTML/CSS

### Archive Database

A lore-driven knowledge archive containing system articles and world records.

### Hidden Protocols

The system contains secret interactions and unlockable content for explorers willing to investigate deeper.

## Installation

### Clone Repository

```bash
git clone https://github.com/V0IDNETWORK/Initial-commit-NEXUS-VOID-2089-demo.git
```

### Enter Project

```bash
cd Initial-commit-NEXUS-VOID-2089-demo
```

### Install Dependencies

```bash
flutter pub get
```

### Run

```bash
flutter run
```

### Run on Web

```bash
flutter run -d chrome
```
